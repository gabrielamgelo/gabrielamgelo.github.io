HR Analytics Dashboard

This project features an interactive dashboard made in Power BI for analyzing HR analytics data. The first page provides insights into the turnover rate, the second page focuses on employee demographics, and the third page displays employee performance metrics. The design of this dashboard was made using Figma.

The dashboard contains:

  • Interactive card visuals that display the total number of hires, terminations, active employees, average employee satisfaction average absences, and average salaries over the 3 pages.
  • A gauge visual for turnover rate.
  • Graphs that display the number of hires and terminations per year, the reason for termination, turnover rate by years at company, average salary by performance, recruitment source, turnover rate by age and more.

Insights:

  • Average salary by department or position.
  • Turnover rate for every department, position, gender, ethnicity and age group.
  • Performance score, satisfaction and absences of each employee.
  • The overall diversity profile of the organization.
  • The best recruiting sources.
